Get Started in Minutes

Three simple steps to your complete PHP development environment

1

Clone Repository

Download IUM from GitHub and navigate to the directory

2

Wake Environment

Start all services with a single command

3

Start Developing

Create your project and access it via HTTPS

Terminal
$ git clone https://github.com/imerfanahmed/ium.git
$ cd ium
$ ./ium wake
$ ./ium composer create-project laravel/laravel my-app
$ ./ium health
🌐 https://my-app.local
